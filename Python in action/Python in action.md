# 5.5 Decimal number to Roman - –®–∞–≥ 1

–í —Ä–∏–º—Å–∫–æ–π —Å–∏—Å—Ç–µ–º–µ —Å—á–∏—Å–ª–µ–Ω–∏—è –¥–ª—è –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏—è —á–∏—Å–µ–ª –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Å–ª–µ–¥—É—é—â–∏–µ —Å–∏–º–≤–æ–ª—ã (—Å–ø—Ä–∞–≤–∞ –∑–∞–ø–∏—Å–∞–Ω—ã —á–∏—Å–ª–∞, –∫–æ—Ç–æ—Ä—ã–º –æ–Ω–∏ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç –≤ –¥–µ—Å—è—Ç–∏—á–Ω–æ–π —Å–∏—Å—Ç–µ–º–µ —Å—á–∏—Å–ª–µ–Ω–∏—è):

I = 1
V = 5
X = 10
L = 50
C = 100
D = 500
M = 1000

–ë—É–¥–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤–∞—Ä–∏–∞–Ω—Ç, –≤ –∫–æ—Ç–æ—Ä–æ–º —á–∏—Å–ª–∞ 4, 9, 40, 90, 400 –∏ 900 –∑–∞–ø–∏—Å—ã–≤–∞—é—Ç—Å—è –∫–∞–∫ –≤—ã—á–∏—Ç–∞–Ω–∏–µ –∏–∑ –±–æ–ª—å—à–µ–≥–æ —á–∏—Å–ª–∞ –º–µ–Ω—å—à–µ–≥–æ: IV, IX, XL, XC, CD –∏ CM, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ.

**–§–æ—Ä–º–∞—Ç –≤–≤–æ–¥–∞**:

–°—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è –Ω–∞—Ç—É—Ä–∞–ª—å–Ω–æ–µ —á–∏—Å–ª–æ n, 0 < n < 4000.

**–§–æ—Ä–º–∞—Ç –≤—ã–≤–æ–¥–∞**:

–°—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è —á–∏—Å–ª–æ, –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –≤ —Ä–∏–º—Å–∫–æ–π —Å–∏—Å—Ç–µ–º–µ —Å—á–∏—Å–ª–µ–Ω–∏—è.

**Sample Input 1**:

1984

**Sample Output 1**:

MCMLXXXIV

**Sample Input 2**:

9

**Sample Output 2**:

IX

**Sample Input 3**:

3

**Sample Output 3**:

III

```
inp = str(input())
result = ''
romans = (
    ('C', 'CC', 'CCC', 'CD', 'D', 'DC', 'DCC', 'DCCC', 'CM', ''),
    ('X', 'XX', 'XXX', 'XL', 'L', 'LX', 'LXX', 'LXXX', 'XC', ''),
    ('I', 'II', 'III', 'IV', 'V', 'VI', 'VII', 'VIII', 'IX', '')
)

for i in range(len(inp)):
    if i == 0 and len(inp) == 4:
        result += 'M' * int(inp[i])
    else:
        result += romans[(i - len(inp)) % 3][int(inp[i]) - 1]

print(result)
```

# 5.4 Roman number to decimal

–í —Ä–∏–º—Å–∫–æ–π —Å–∏—Å—Ç–µ–º–µ —Å—á–∏—Å–ª–µ–Ω–∏—è –¥–ª—è –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏—è —á–∏—Å–µ–ª –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Å–ª–µ–¥—É—é—â–∏–µ —Å–∏–º–≤–æ–ª—ã (—Å–ø—Ä–∞–≤–∞ –∑–∞–ø–∏—Å–∞–Ω—ã —á–∏—Å–ª–∞, –∫–æ—Ç–æ—Ä—ã–º –æ–Ω–∏ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—Ç –≤ –¥–µ—Å—è—Ç–∏—á–Ω–æ–π —Å–∏—Å—Ç–µ–º–µ —Å—á–∏—Å–ª–µ–Ω–∏—è):

I = 1
V = 5
X = 10
L = 50
C = 100
D = 500
M = 1000

–ë—É–¥–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤–∞—Ä–∏–∞–Ω—Ç, –≤ –∫–æ—Ç–æ—Ä–æ–º —á–∏—Å–ª–∞ 4, 9, 40, 90, 400 –∏ 900 –∑–∞–ø–∏—Å—ã–≤–∞—é—Ç—Å—è –∫–∞–∫ –≤—ã—á–∏—Ç–∞–Ω–∏–µ –∏–∑ –±–æ–ª—å—à–µ–≥–æ —á–∏—Å–ª–∞ –º–µ–Ω—å—à–µ–≥–æ: IV, IX, XL, XC, CD –∏ CM, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ.

–ù–∞–ø–∏—à–∏—Ç–µ –ø—Ä–æ–≥—Ä–∞–º–º—É, –∫–æ—Ç–æ—Ä–∞—è –ø–µ—Ä–µ–≤–æ–¥–∏—Ç —á–∏—Å–ª–æ –∏–∑ —Ä–∏–º—Å–∫–æ–π –≤ –¥–µ—Å—è—Ç–∏—á–Ω—É—é —Å–∏—Å—Ç–µ–º—É —Å—á–∏—Å–ª–µ–Ω–∏—è.

**–§–æ—Ä–º–∞—Ç –≤–≤–æ–¥–∞**:

–°—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è —á–∏—Å–ª–æ, –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –≤ —Ä–∏–º—Å–∫–æ–π —Å–∏—Å—Ç–µ–º–µ —Å—á–∏—Å–ª–µ–Ω–∏—è. –ì–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç—Å—è, —á—Ç–æ —á–∏—Å–ª–æ –º–µ–Ω—å—à–µ 4000.

**–§–æ—Ä–º–∞—Ç –≤—ã–≤–æ–¥–∞**:

–°—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è —á–∏—Å–ª–æ –≤ –¥–µ—Å—è—Ç–∏—á–Ω–æ–π —Å–∏—Å—Ç–µ–º–µ —Å—á–∏—Å–ª–µ–Ω–∏—è, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–µ –≤–≤–µ–¥—ë–Ω–Ω–æ–º—É.

**Sample Input 1**:

MCMLXXXIV

**Sample Output 1**:

1984

**Sample Input 2**:

IX

**Sample Output 2**:

9

**Sample Input 3**:

III

**Sample Output 3**:

3

```
from collections import OrderedDict as od
roman = input()
decimal = 0

d = (
        (900, 'CM'),
        (1000, 'M'),
        (400, 'CD'),
        (500, 'D'),
        (90, 'XC'),
        (100, 'C'),
        (40, 'XL'),
        (50, 'L'),
        (9, 'IX'),
        (10, 'X'),
        (4, 'IV'),
        (5, 'V'),
        (1, 'I')
    )

d = od(d)

for k in d:
    if roman.find(d[k]) >= 0:
        decimal += k * roman.count(d[k])
        roman = roman.replace(d[k], '')

print(decimal)
```

# 5.2 Old LCD calculator

–ù–∞–ø–∏—à–∏—Ç–µ –ø—Ä–æ–≥—Ä–∞–º–º—É, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–≤–æ–¥–∏—Ç —á–∏—Å–ª–æ –≤ —Å—Ç–∏–ª–µ LCD –∫–∞–ª—å–∫—É–ª—è—Ç–æ—Ä–∞.

# 5.1 Collatz conjecture or the 3n + 1 problem - –®–∞–≥ 1

–ù–∞–ø–∏—à–∏—Ç–µ —Ñ—É–Ω–∫—Ü–∏—é, –∫–æ—Ç–æ—Ä–∞—è –¥–ª—è –∑–∞–¥–∞–Ω–Ω–æ–≥–æ –Ω–∞—Ç—É—Ä–∞–ª—å–Ω–æ–≥–æ —á–∏—Å–ª–∞ n –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —á–∏—Å–µ–ª, –æ–ø–∏—Å–∞–Ω–Ω—É—é –≤ –≥–∏–ø–æ—Ç–µ–∑–µ –ö–æ–ª–ª–∞—Ç—Ü–∞:

–ï—Å–ª–∏ n —á–µ—Ç–Ω–æ–µ, —Ç–æ –¥–µ–ª–∏–º –µ–≥–æ –ø–æ–ø–æ–ª–∞–º, –µ—Å–ª–∏ –Ω–µ—á—ë—Ç–Ω–æ–µ, —Ç–æ —É–º–Ω–æ–∂–∞–µ–º –Ω–∞ 3 –∏ –ø—Ä–∏–±–∞–≤–ª—è–µ–º 1. –° –∏—Ç–æ–≥–æ–º –≤—ã—á–∏—Å–ª–µ–Ω–∏—è —Å–Ω–æ–≤–∞ –ø—Ä–æ–¥–µ–ª—ã–≤–∞–µ–º —ç—Ç—É –æ–ø–µ—Ä–∞—Ü–∏—é –¥–æ —Ç–µ—Ö –ø–æ—Ä, –ø–æ–∫–∞ –≤ —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ –Ω–µ –±—É–¥–µ—Ç –ø–æ–ª—É—á–µ–Ω–æ —á–∏—Å–ª–æ 1.

–ù–∞–ø—Ä–∏–º–µ—Ä, –¥–ª—è —á–∏—Å–ª–∞ n = 17 –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –≤—ã—á–∏—Å–ª–µ–Ω–∏–π –≤—ã–≥–ª—è–¥–∏—Ç —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:
17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1

–ü—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ—Ç—Å—è, —á—Ç–æ –ø–æ–¥–æ–±–Ω–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –æ—Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –Ω–∞ —á–∏—Å–ª–µ 1 –¥–ª—è –ª—é–±–æ–≥–æ –Ω–∞—á–∞–ª—å–Ω–æ–≥–æ –Ω–∞—Ç—É—Ä–∞–ª—å–Ω–æ–≥–æ —á–∏—Å–ª–∞ n. 

**–§–æ—Ä–º–∞—Ç –≤–≤–æ–¥–∞**:

–°—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–æ–µ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ n, n > 0.

**–§–æ—Ä–º–∞—Ç –≤—ã–≤–æ–¥–∞**:

–°—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª, —Ä–∞–∑–¥–µ–ª—ë–Ω–Ω—ã—Ö –ø—Ä–æ–±–µ–ª–æ–º.

**Sample Input 1**:

17

**Sample Output 1**:

17 52 26 13 40 20 10 5 16 8 4 2 1

**Sample Input 2**:

1

**Sample Output 2**:

1

```
n = int(input())

def collatz(n):
    if n == 1:
        print(n, end = " ")
        return 1
    elif n % 2 == 0:
        print (n, end = " ")
        return collatz(n / 2)
    else:
        print (n, end = " ")
        return collatz(n * 3 + 1)

collatz(n)
```

–í—ã–≤–µ–¥–∏—Ç–µ —Ç–∞–±–ª–∏—Ü—É —Ä–∞–∑–º–µ—Ä–æ–º n √ó n, –∑–∞–ø–æ–ª–Ω–µ–Ω–Ω—É—é —Ü–µ–ª—ã–º–∏ —á–∏—Å–ª–∞–º–∏ –æ—Ç 1 –¥–æ n**2 –ø–æ —Å–ø–∏—Ä–∞–ª–∏, –≤—ã—Ö–æ–¥—è—â–µ–π –∏–∑ –ª–µ–≤–æ–≥–æ –≤–µ—Ä—Ö–Ω–µ–≥–æ —É–≥–ª–∞ –∏ –∑–∞–∫—Ä—É—á–µ–Ω–Ω–æ–π –ø–æ —á–∞—Å–æ–≤–æ–π —Å—Ç—Ä–µ–ª–∫–µ, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ –ø—Ä–∏–º–µ—Ä–µ.

**–§–æ—Ä–º–∞—Ç –≤–≤–æ–¥–∞**:
–û–¥–Ω–∞ —Å—Ç—Ä–æ–∫–∞, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è –æ–¥–Ω–æ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ n, n > 0.

**–§–æ—Ä–º–∞—Ç –≤—ã–≤–æ–¥–∞**:
–¢–∞–±–ª–∏—Ü–∞ –∏–∑ n —Å—Ç—Ä–æ–∫, –∑–Ω–∞—á–µ–Ω–∏—è –≤ —Å—Ç—Ä–æ–∫–∞—Ö —Ä–∞–∑–¥–µ–ª–µ–Ω—ã –ø—Ä–æ–±–µ–ª–æ–º.

**Sample Input**:

5

**Sample Output**:

```
1 2 3 4 5
16 17 18 19 6
15 24 25 20 7
14 23 22 21 8
13 12 11 10 9
```
```
n = int(input())
a = [[0 for x in range(n)] for x in range(n)]
counter = 1
offset = 0

def right(i):
    global counter, offset
    j = offset
    while j < n - i and a[i][j] == 0:
        a[i][j] = counter
        j += 1
        counter += 1
    offset += 1

def down(i):
    global counter, offset
    j = offset
    while j < n - i and a[j][n - 1 - i] == 0:
        a[j][n - 1 - i] = counter
        j += 1
        counter += 1

def left(i):
    global counter, offset
    j = offset
    while j < n - i and a[n - 1 - i][n - 1 - j] == 0:
        a[n - 1 - i][n - 1 - j] = counter
        j += 1
        counter += 1

def up(i):
    global counter, offset
    j = offset
    while j < n - i and a[n - 1 - j][i] == 0:
        a[n - 1 - j][i] = counter
        j += 1
        counter += 1

for i in range(n):
    right(i)
    down(i)
    left(i)
    up(i)

for e in a:
    print(" ".join(map(lambda x: str(x), e)))
```

# 3.4 Modify list

–ù–∞–ø–∏—à–∏—Ç–µ —Ñ—É–Ω–∫—Ü–∏—é modify_list(l), –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –Ω–∞ –≤—Ö–æ–¥ —Å–ø–∏—Å–æ–∫ —Ü–µ–ª—ã—Ö —á–∏—Å–µ–ª, —É–¥–∞–ª—è–µ—Ç –∏–∑ –Ω–µ–≥–æ –≤—Å–µ –Ω–µ—á—ë—Ç–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è, –∞ —á—ë—Ç–Ω—ã–µ –Ω–∞—Ü–µ–ª–æ –¥–µ–ª–∏—Ç –Ω–∞ –¥–≤–∞. –§—É–Ω–∫—Ü–∏—è –Ω–µ –¥–æ–ª–∂–Ω–∞ –Ω–∏—á–µ–≥–æ –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å, —Ç—Ä–µ–±—É–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ –∏–∑–º–µ–Ω–µ–Ω–∏–µ –ø–µ—Ä–µ–¥–∞–Ω–Ω–æ–≥–æ —Å–ø–∏—Å–∫–∞

```
def modify_list(l):
    idx = 0
    while idx < len(l):
        if l[idx] % 2:
            l.pop(idx)
        else:
            l[idx] = l[idx] // 2
            idx += 1
```
