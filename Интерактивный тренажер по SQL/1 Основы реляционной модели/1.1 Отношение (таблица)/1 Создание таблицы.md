# Создание таблицы

## Задание

Сформулируйте SQL запрос для создания таблицы **book**. Структура таблицы **book**:

| Поле | Тип, описание |
|---|---|
| book_id | INT PRIMARY KEY AUTO_INCREMENT |
| title | VARCHAR(50) |
| author | VARCHAR(30) |
| price | DECIMAL(8, 2) |
| amount | INT |

## Решение

```
CREATE TABLE book(
    book_id INT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(50),
    author VARCHAR(30),
    price DECIMAL(8, 2),
    amount INT
);
```
